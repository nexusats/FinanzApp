# FinanzApp - Actividad 4

Prototipo web académico desarrollado con HTML5, CSS3, JavaScript y Bootstrap.

## Archivos principales

- `index.html`: estructura del sitio.
- `styles.css`: diseño visual y responsive.
- `script.js`: interacción básica para registrar movimientos simulados.

## Publicación recomendada en GitHub Pages

1. Crear un repositorio público o privado en GitHub.
2. Subir `index.html`, `styles.css` y `script.js` en la raíz del repositorio.
3. Entrar a `Settings > Pages`.
4. En `Build and deployment`, seleccionar `Deploy from a branch`.
5. Elegir la rama `main` y carpeta `/root`.
6. Guardar y esperar a que GitHub genere la URL pública.
7. Copiar la URL generada en el informe de entrega.

## Publicación alternativa en Netlify

1. Entrar a Netlify Drop.
2. Arrastrar la carpeta del proyecto.
3. Esperar la publicación automática.
4. Copiar la URL generada en el informe.

## Video

El sitio trae un video de ejemplo. Debe reemplazarse por el video real del estudiante.

En `index.html`, buscar:

```html
<iframe src="about:blank"
```

Y reemplazar `about:blank` por el enlace embebido del video real, por ejemplo `https://www.youtube.com/embed/ID_DEL_VIDEO`.
