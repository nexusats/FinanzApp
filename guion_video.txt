Guion sugerido para el video demostrativo - FinanzApp Actividad 4

1. Presentación inicial
Buenos días, mi nombre es Jorge Ivan Usuga Romero. En este video presento el desarrollo del prototipo web de FinanzApp, una aplicación orientada al control de finanzas personales.

2. Contexto del proyecto
El proyecto nace de la necesidad de organizar ingresos, gastos, presupuestos, metas de ahorro y deudas en una sola solución digital. En la actividad anterior se definió el wireframe y la estructura general; en esta actividad se implementa una versión programada del prototipo.

3. Tecnologías utilizadas
El prototipo fue desarrollado con HTML5 para la estructura, CSS3 para el diseño visual, JavaScript para la interacción básica y Bootstrap para facilitar la construcción responsiva.

4. Recorrido por el sitio
En la página inicial se presenta el objetivo de FinanzApp. Luego se muestran el problema, los módulos principales, una demo interactiva del usuario y una vista del panel administrativo web.

5. Demostración funcional
En la demo se puede registrar un movimiento de ingreso o gasto. Al ingresar el valor, el sistema actualiza el balance estimado, los ingresos, los gastos y la lista de últimos movimientos.

6. Panel administrativo
El panel administrativo representa la parte web propuesta para Laravel. Allí se observan indicadores, tabla de movimientos, usuarios, categorías y acciones de exportación.

7. Hosting
El proyecto fue publicado en un hosting gratuito, preferiblemente GitHub Pages o Netlify. La URL pública se incluye en el informe final.

8. Cierre
Con este desarrollo se demuestra la transición del wireframe inicial hacia una versión web funcional, manteniendo la misma idea de FinanzApp y preparando el proyecto para futuras fases de desarrollo.
